# คู่มือการทำงานสำหรับนักศึกษาสหกิจ

## ระยะเวลา

1 สัปดาห์

## เป้าหมาย

พัฒนาและปรับปรุงระบบ ESP32 Web Flasher

## สิ่งที่ต้องส่งมอบ

- Source Code
- README ที่อัปเดต
- Pull Request
- Demo การทำงาน

## ขั้นตอนการทำงาน

1. รับ Task
2. สร้าง Branch
3. พัฒนา Feature
4. Commit Code
5. เปิด Pull Request
6. Code Review
7. Merge

## กฎการทำงาน

- ทุกคนต้องมีส่วนร่วม
- ทุกคนต้องมี Commit ของตนเอง
- ทุกคนต้องเปิด Pull Request อย่างน้อย 1 ครั้ง
- ทุกคนต้องช่วย Review งานเพื่อน
- ห้าม Commit เข้า main โดยตรง

- วันนี้จะทำอะไร
- ติดปัญหาอะไร
