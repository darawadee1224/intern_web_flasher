# ESP32 Web Flasher

## รายละเอียดโครงการ

ESP32 Web Flasher เป็นเครื่องมือสำหรับแฟลชเฟิร์มแวร์ลงบอร์ด ESP32 ผ่านเว็บเบราว์เซอร์ โดยใช้ Web Serial API ร่วมกับไลบรารี `esptool-js` ทำให้สามารถเชื่อมต่อพอร์ตซีเรียล, อ่านข้อมูลชิป, ลบแฟลช, ได้จากอินเทอร์เฟซบนเว็บ

## ฟังก์ชันหลัก

- เชื่อมต่อกับ ESP32 ผ่าน `navigator.serial.requestPort()`
- เลือกค่า Baud Rate ได้จาก UI
- แสดงสถานะการเชื่อมต่อและข้อมูลชิป
- แสดง progress bar แบบเรียลไทม์ขณะเขียนแฟลช
- ดาวน์โหลด Log เป็น `.txt`
- ล้างข้อความ Log

## วิธีใช้งาน

1. เปิดเว็บอินเทอร์เฟซของโปรเจกต์
2. เลือก Baud Rate ที่ต้องการ
3. กดปุ่ม `CONNECT PORT` เพื่อเลือกพอร์ต Serial ของ ESP32
4. กดปุ่ม `DISCONNECT` เพื่อออกจากการเชื่อมต่อพอร์ต
5. หากต้องการบันทึกข้อมูลการทำงาน ให้กด `DOWNLOAD LOG`
6. หากต้องการล้างหน้าจอ Log ให้กด `CLEAR`

## วิธีติดตั้ง

1. Clone repository
2. ติดตั้ง dependencies

```bash
npm install
```

3. รันโปรแกรม

```bash
npm run dev
```

## โครงสร้างโปรเจกต์

- `web/index.html` - หน้า UI ของแอปพลิเคชัน
- `web/flash.js` - โค้ดควบคุมการเชื่อมต่อ, แฟลช, และ Log
- `web/style.css` - รูปแบบและการจัดวางหน้าเว็บ

## เทคโนโลยีที่ใช้

- Web Serial API
- esptool-js
- HTML/CSS/JavaScript
