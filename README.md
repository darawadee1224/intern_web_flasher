# ESP32 Web Flasher

## รายละเอียดโครงการ

เครื่องมือสำหรับอัปโหลด Firmware ไปยัง ESP32 ผ่าน Web Browser โดยใช้ Web Serial API

## วัตถุประสงค์

- ศึกษาการทำงานของ Frontend
- ฝึกการทำงานร่วมกันผ่าน GitHub
- ฝึกการจัดการงานแบบ Agile
- พัฒนาเครื่องมือที่สามารถนำไปใช้งานจริงได้

## วิธีติดตั้ง

1. Clone Repository

2. ติดตั้ง Package

npm install

3. รันโปรแกรม

npm run dev
