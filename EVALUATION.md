# เกณฑ์การประเมิน

คะแนนรวม 100 คะแนน

## 1. การทำงานเป็นทีม (30 คะแนน)

- การสื่อสาร
- การช่วยเหลือเพื่อนร่วมทีม
- การรับผิดชอบงาน

## 2. การใช้ Git และ GitHub (20 คะแนน)

- Branch
- Commit
- Pull Request
- Code Review

## 3. การจัดการงาน (20 คะแนน)

- รับผิดชอบ Task
- ส่งงานตรงเวลา
- อัปเดตสถานะงาน

## 4. เอกสาร (15 คะแนน)

- README
- Comment
- เอกสารประกอบ

## 5. ผลลัพธ์ของงาน (15 คะแนน)

- ฟังก์ชันทำงานได้
- คุณภาพของงาน
- การทดสอบ
